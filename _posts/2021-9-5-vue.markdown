install VUE
```
# 创建一个基于 webpack 模板的新项目
$ vue init webpack my-project
# 这里需要进行一些配置，默认回车即可
This will install Vue 2.x version of the template.

$ cd my-project
$ cnpm install
// 运行VUE
$ cnpm run dev
```

npm
```
# 全局安装
npm install -g webpack

# 全局删除
npm uninstall -g webpack

# 查看npm安装module版本
npm list

# 查看单个npm库的version
npm view <package> version
```
