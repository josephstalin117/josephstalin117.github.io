# Claude code 使用


## 常用命令
```
# 清空上下文
/clear

# 弹出最近的会话
/resume

# 项目初始化
/init

# 临时插入对话
/btw
```

## 上下文管理
```
# 查看上下文占用
/context

# 上下文占用常驻显示
/statusline

# 压缩上下文
/compact

# 项目分叉, 延两种不同的方向实现
/branch
```

## 项目验收

```
# 查看修改内容
/diff

# 代码审查
/simplify

# 获取历史会话/回退代码
/rewind
```
