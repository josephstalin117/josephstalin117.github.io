基本命令
```
重新配置：gitlab-ctl reconfigure
启动：gitlab-ctl start
重新启动：gitlab-ctl restart
停止：gitlab-ctl stop
查看状态：gitlab-ctl status
实时日志查看：gitlab-ctl tail
检查报错：gitlab-rake gitlab:check
```

卸载
```
# 停止
gitlab-ctl stop

# Debian/Ubuntu
sudo dpkg -r gitlab-ce

# Redhat/Centos
rpm -e gitlab-ce

# 
ps aux | grep gitlab
```

更改账户
```
sudo gitlab-rake "gitlab:password:reset[johndoe]"
```

docker中停止gitlab服务器
```
# 先进入gitlab镜像中
docker exec -it gitlab_instance bash
# 停止gitlab服务器
gitlab-ctl stop
exit
```
